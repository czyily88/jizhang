- **修复乱码**: 主界面条目中付款方式字段显示名称而非 ID

## 版本规范

版本号格式：`X.Y.Z`

| 类型 | 说明 | 示例 |
|------|------|------|
| **major** | 重大改动，向后不兼容 | 1.0.0 → 2.0.0 |
| **minor** | 新功能，向后兼容 | 1.0.0 → 1.1.0 |
| **patch** | Bug 修复，向后兼容 | 1.0.0 → 1.0.1 |

### 使用命令
```bash
# 查看当前版本
version.bat

# 交互式升级
version.bat bump

# 直接升级对应版本
version.bat major   # 主版本号 +1
version.bat minor   # 次版本号 +1  
version.bat patch   # 修订号 +1

# 构建 APK（会自动显示当前版本）
version.bat build
```
